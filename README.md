# Tiến Lên Miền Nam

Chào mừng bạn đến với dự án **Tiến Lên Miền Nam** - một ứng dụng game bài di động được phát triển trên nền tảng Android.

## 📖 Giới thiệu

Đây là dự án mô phỏng trò chơi bài đánh bài Tiến Lên Miền Nam phổ biến tại Việt Nam. Ứng dụng tập trung vào trải nghiệm người dùng với giao diện thân thiện và các tính năng tương tác hấp dẫn.

## ✨ Tính năng nổi bật

Dựa trên cấu trúc dự án hiện tại, ứng dụng bao gồm các tính năng:

*   **Chơi bài Tiến Lên:** Luật chơi miền Nam truyền thống.
*   **Hệ thống phòng chơi:**
    *   Tính năng mời bạn bè.
*   **Sự kiện:**
    *   Hệ thống nhiệm vụ (Quest).
    *   Giao diện sự kiện độc đáo.
*   **Giao diện tùy chỉnh:** Sử dụng các thành phần UI được thiết kế riêng.

## 🛠 Công nghệ sử dụng

*   **Ngôn ngữ lập trình:** Java
*   **IDE:** Android Studio
*   **UI:** XML Layouts & Custom Views
*   **Build System:** Gradle

## ⚙️ Hướng dẫn cài đặt

Để chạy dự án này trên máy cục bộ của bạn, vui lòng làm theo các bước sau:

### 1. Yêu cầu hệ thống
*   [Android Studio](https://developer.android.com/studio) (Phiên bản mới nhất được khuyến nghị).
*   JDK 11 trở lên.
*   Thiết bị Android hoặc máy ảo (Emulator) chạy Android 7.0 trở lên (API level 24+).

### 2. Các bước cài đặt

1.  **Sao chép (Clone) dự án:**
    ```bash
    git clone <đường-dẫn-repo-của-bạn>
    ```
    Hoặc tải xuống file ZIP và giải nén.

2.  **Mở dự án trong Android Studio:**
    *   Khởi động Android Studio.
    *   Chọn **File** -> **Open...**
    *   Điều hướng đến thư mục `Tienlenmiennam` và nhấn OK.

3.  **Đồng bộ hóa Gradle:**
    *   Đợi Android Studio tải các thư viện cần thiết và lập chỉ mục (index) các tệp tin.
    *   Nếu có thông báo lỗi, hãy thử nhấn vào nút **Try Again** hoặc **Sync Project with Gradle Files**.

4.  **Chạy ứng dụng:**
    *   Kết nối thiết bị thật qua USB hoặc khởi động máy ảo Android.
    *   Nhấn nút **Run** (biểu tượng tam giác xanh lá) trên thanh công cụ.
